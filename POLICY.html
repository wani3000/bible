KorRV 성경 앱 초안 정책서

시행일: 2026-02-19 (초안)

본 정책서는 현재 개발된 프로토타입의 운영 기준을 설명합니다. 실제 서비스 배포 전 법률 검토 후 확정본으로 교체되어야 합니다.

1. 서비스 범위

  • 본 앱은 성경 읽기, 읽기 진행 저장, 그룹 진행 확인 기능을 제공합니다.
  • 현재 버전은 프로토타입으로 일부 샘플 데이터만 포함됩니다.
  • 서비스 기능은 사전 공지 없이 변경될 수 있습니다.

2. 콘텐츠 및 저작권 정책

  • 현재 본문은 KorRV 구조 참고 샘플 데이터입니다.
  • 정식 본문 연동 시, 역본별 라이선스 및 저작권 정책을 준수해야 합니다.
  • 저작권 보호 역본(예: 개역개정 등)은 권리자 승인 없이 배포하지 않습니다.
  • 콘텐츠 출처, 라이선스, 저작권 고지는 앱 내에 명시합니다.

3. 개인정보 처리(현재 구현 기준)

  • 현재 프로토타입은 서버 전송 없이 브라우저 로컬 저장소(LocalStorage)만 사용합니다.
  • 저장 항목: bible.progress, bible.groupMembers, bible.readerFont
  • 사용자가 브라우저 데이터를 삭제하면 저장 정보도 함께 삭제됩니다.
  • 추후 계정/클라우드 기능 도입 시 별도 개인정보처리방침을 갱신합니다.

4. 그룹 기능 운영 정책

  • 그룹 내 사용자 읽기 진행 정보는 그룹 구성원에게 표시될 수 있습니다.
  • 사용자 초대/삭제 기능은 그룹 관리 목적에 한해 사용해야 합니다.
  • 타인의 식별정보를 무단 수집/공유하는 행위를 금지합니다.

5. 폰트 및 UI 자산 정책

  • 앱 전체 기본 서체: Pretendard
  • 본문 선택 서체: Nanum Myeongjo
  • UI 컴포넌트: FlyonUI (MIT)
  • 각 서체/라이브러리의 라이선스 조건을 준수합니다.

6. 탈퇴/데이터 삭제 정책(초안)

  • 현재 버전의 탈퇴 버튼은 UI 초안 상태이며 서버 계정 삭제는 아직 미연동입니다.
  • 로컬 데이터 삭제는 브라우저 저장소 삭제로 수행할 수 있습니다.
  • 서버 계정 기능이 도입되면 계정 삭제 절차와 보관기간을 별도 명시합니다.

7. 면책 및 한계

  • 본 앱은 종교적/학습 보조 도구이며, 공식 성경 출판물의 대체를 보장하지 않습니다.
  • 프로토타입 단계에서 데이터 누락/오류가 발생할 수 있습니다.
  • 법적 분쟁 방지를 위해 배포 전 라이선스 검토를 우선합니다.
