KorRV 성경 앱 초안 README

이 문서는 현재 저장소에 구현된 성경 앱 프로토타입의 기능, 구조, 실행 방법을 정리합니다.

Next.js 14 React 18 Tailwind CSS + FlyonUI LocalStorage 기반 상태 저장

1. 현재 구현 범위

하단 탭 3개

  • 홈: 오늘 날짜, 다음 읽을 장/절 안내, 일독표 섹션, 그룹 진행 현황
  • 성경읽기: 구약/신약 + 책 선택, 본문 페이지 이동, 여기까지 읽기
  • 마이: 계정 표시, 그룹 초대/삭제, 약관/탈퇴 버튼(초안)

설정 기능

  • 마이 탭 우상단 설정 아이콘 진입
  • 성경 본문 서체 선택: Pretendard / Nanum Myeongjo
  • 선택값 로컬 저장: bible.readerFont

2. 데이터/저장 정책(현재)

  • 성경 본문은 KorRV 구조 참고 샘플 데이터만 포함
  • 읽기 진행도 저장: bible.progress
  • 그룹 샘플 저장: bible.groupMembers
  • 본문 서체 저장: bible.readerFont
  • 서버 DB/Firebase 연동은 아직 미적용

3. 주요 파일

  • src/app/page.tsx: 앱 메인 화면(탭/읽기/마이/설정)
  • src/data/bibleKorRvMock.ts: KorRV 샘플 본문 데이터
  • src/app/layout.tsx: 전역 폰트 로딩(나눔명조 포함)
  • tailwind.config.mjs: FlyonUI 플러그인 설정

4. 로컬 실행 방법

npm install
npm run dev

프로덕션 빌드 확인:

npm run build

5. 다음 개발 권장 항목

  • KorRV 전체 본문 파서 연동(USFM/SWORD 입력)
  • 일독표 로직(30/90/180/365일 플랜) 고도화
  • 그룹 기능 서버 연동(Firebase Auth + Firestore)
  • 약관/개인정보 페이지 실제 문서 연결
